# ACC102 Assignment: AAPL vs MSFT Stock Analysis

## 1. Project Overview
This project analyzes and compares the stock performance of **Apple (AAPL)** and **Microsoft (MSFT)** from **2023-01-01 to 2025-01-01**.
Data is retrieved from the **WRDS CRSP database** and includes return analysis, risk metrics, and visualization.

---

## 2. Data Source
- Database: WRDS CRSP Monthly / Daily Stock File
- Stocks:
  - AAPL: permno = 14593
  - MSFT: permno = 10107
- Time period: 2023-01-01 to 2025-01-01

---

## 3. Key Metrics Calculated
- Daily Price and Daily Return
- Descriptive Statistics (mean, std, min, max, etc.)
- Annualized Return
- Annualized Volatility
- Sharpe Ratio (risk-free rate = 2%)

---

## 4. Visualizations (4 Subplots)
1. Price Trend Comparison
2. Daily Return Distribution
3. Cumulative Return
4. Sharpe Ratio Comparison

---

## 5. How to Run
1. Open the `.ipynb` file in Jupyter / VS Code / Colab
2. Replace `username` with your WRDS account
3. Run all cells: **Run All**
4. All outputs, statistics, and figures will be generated automatically

---

- `ACC102_Assignment_AAPL_MSFT.ipynb`: Main code
- `README.md`: Project description
- Outputs: Statistics, charts, performance summary
